# Luxe Films

A luxury wedding filmmaker portfolio website built with Next.js, Tailwind CSS and Framer Motion.

## Live Demo

https://luxe-films.vercel.app

## Features

* Cinematic Hero Section
* Featured Wedding Film
* Dynamic Story Pages
* Luxury Gallery Layout
* Responsive Design
* WhatsApp Integration
* Contact Section
* Vercel Deployment
* Speed Insights

## Tech Stack

* Next.js
* React
* Tailwind CSS
* Framer Motion
* Vercel
